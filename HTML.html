Why is HTML a must for every Website

HTML stands for Hyper Text Markup Language. It is used to design web pages using the markup language.
A hypertext is a text that is used to reference other pieces of text, while a markup language is a series of markings
that tells web servers the style and structure of a document. HTML is used to create the structure of web pages that
are displayed on the World Wide Web (www). It contains Tags and Attributes that are used to design the web pages.

There are several ways to classify HTML tags, such as:

Block-level elements and Inline elements
container and empty tags

HTML uses predefined tags that tell the browser how to display the content. Tags are nothing but some instructions
that are enclosed in angle braces (i.e., <>). There are two types of tags in HTML:

  • Container Tags
  • Empty tags

Container Tags

Container tags are divided into three parts, i.e., opening tag, content(which will display on the browser), and closing tag.
In the content part, they can also contain some other tags. These opening and closing tags are used in pairs which are start tag and end tag.

The following are the Syntax for container tags:

  • <html>...</html>: This marks the beginning and ending of the webpage

  • <head>...</head>: This tag is used to define the head part of the document

  • <title>...</title>:This tag stores the description of the web page

  • <body>...</body>:This tag is used to display all the information on the webpage to the user.

  • <h1>...</h1>to<h6>...</h6>: :It is used for including headings of different sizes ranging from 1 to 6.

  • <p>...</p>:When paragraphs are needed to be included, this tag is used

  • <b>...</b>:Makes the contained text to bold.

  • <i>...</i>:Makes the contained text to italic.

  • <a href=”link.com”>...</a>:This hyperlink is used to navigate to other webpages

  • <button>...</button>:This tag is used to create a click button

  • <div>...</div>:This defines a section in a document. The webpage can be divided to different sections

  • <nav>...</nav>:Defines a navigation bar that contains a set of menu or a menu of hyperlinks.

  • <script>...</script>:This contains the javascript code that adds interactivity to the webpage.

  • <ol>...</ol>:This tag is used to create ordered lists.

  • <ul>...</ul>:This tag is used to create unordered lists.

  • <li>...</li>:This tag is used to add list items.

  • <span>...</span>:A container for in-line content, such as content inside a paragraph.

  • <em>...</em>:Gives the contained text emphasis (usually as italics).

  • <strong>...</strong>:Makes the contained text bold.

Empty Tags

The tags that do not contain any closing tags are known as empty tags. Empty tags contain only the opening tag but they perform some action in the webpage.

The following are the common Syntax for Empty tags:

  • <br>:Inserts a line break in a webpage wherever needed.

  • <hr>:Inserts a horizontal line wherever needed in the webpage.

  • <img>:This tag is used to display the images on the webpage which were given in the src attribute of the tag.

  • <input>:This is mainly used with forms to take the input from the user and we can also define the type of the input.

  • <link>:When we store our CSS in an external file this can be used to link external files and documents to the webpage and it is mainly used to link CSS files.

  • <meta>:Contains all metadata of the webpage. Metadata is the data about data and is described in the head tag.

  • <source>:When an external media source is needed to be included in the webpage. source tag is used to insert any media source like audio, video etc… in our webpage.

Other less commonly used syntax for empty tags include:

  • <area>

  • <base>

  • <col>

  • <embed>

  • <keygen>

  • <param>

  • <track>

  • <wbr>
